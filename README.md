# BissTrainer Mobile

Native Expo-Version fuer den iPhone-Start des Projekts.

## Starten

1. `cd mobile`
2. `npx expo start`
3. Auf dem iPhone `Expo Go` oeffnen
4. Den QR-Code aus dem Terminal oder Browser scannen

## Warum Expo

- Schnellster Weg auf ein physisches iPhone
- Einfache lokale Vorschau mit Expo Go
- Spaeter auf native Builds erweiterbar, ohne die App neu zu denken

## Aktueller Stand

- Fallbibliothek mit 6 Faellen
- Quiz, Punkte und Fortschritt im laufenden App-State
- Touch-optimierte, native Ein-Bildschirm-Oberflaeche

## Naechste Schritte

- Persistenz via AsyncStorage
- Medienmodule pro Fall
- Tutor:innenmodus und echtes Backend-Ranking
